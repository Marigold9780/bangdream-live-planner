# BanG Dream Live Planner

一个用于 BanG Dream（Bushiroad）Live 演出查询、行程规划和机票搜索的 QoderWork 技能。

## 功能概述

本技能支持四种主要模式，可根据用户需求灵活组合：

- **演出信息研究** — 汇总即将举办或已结束的 Live 演出详细报告
- **旅行与航班规划** — 结合假期/ vacation 日期匹配演出，搜索最优往返航班组合
- **演出详情查询** — 深入查询特定演出信息（歌单、场馆、特典、宣传图）
- **地点优先发现** — 用户指定日本城市和时间段，反向搜索该城市附近的 Live 演出并推荐

## 核心特性

### 智能搜索

- 优先精确匹配用户提到的演出名称，不跳过已公布的官方活动去推测未来事件
- 多层级搜索策略：官方标题 → 日文名称 + "日程" → 专辑名 + "リリース記念ライブ"
- 从官方网站、Twitter/X、场馆网站、粉丝 Wiki 等多渠道交叉验证信息

### 票务状态检查

自动查询并标注每张演出门票的销售状态，使用颜色图标醒目提示：

| 状态 | 图标 | 含义 |
|------|------|------|
| 可售中 | 🟢 | 官方渠道正常销售 |
| 即将开售 | 🟡 | 尚未开售，需设置提醒 |
| 刚结束 | 🔴 | 近 2-4 周刚结束，需立即查看二手平台 |
| 已结束 | ⚫ | 销售已结束数月 |
| 售罄 | ⚫ | 官方标注售罄 |
| 未知 | ⚪ | 无法确认状态 |

紧急状态会在报告顶部醒目警告。

### 真实机票搜索

- **必须进行实际网页搜索**，不使用泛泛的价格区间
- 至少搜索 3 个平台（携程、去哪儿、Google Flights、Expedia、Skyscanner 等）
- 提供具体航班号、起降时间、精确价格、行李额度
- 至少提供 3 个选项：最佳性价比、最低价、最舒适
- 对于 3 个月以上的远期航班，明确标注数据限制并提供直链

### 完整行程规划

- 签证准备倒计时清单
- 每日详细时间安排（抵达日、自由日、Live 日、返程日）
- 多方案对比：每个请假方案都有完整的详细日程，不含糊
- 场馆交通路线、入场时间建议、周边活动推荐

### 景点优先级

推荐景点时按以下优先级排序：

1. **P0 — 经典必去**：浅草寺、明治神宫、涩谷十字路口等标志性景点
2. **P1 — 二次元圣地巡礼**：来自 BanG Dream、Girls Band Cry 等作品的现实取景地（参考 [anitabi.cn](https://www.anitabi.cn/)）
3. **P2 — 网络文化地标**：如"下北泽野兽宅邸"等 meme 丰富的地点

### 货币清晰标注

- 人民币使用 `RMB` 前缀（机票、签证等）
- 日元使用 `JPY` 前缀（日本门票、酒店、餐饮、交通等）
- **绝不混合货币直接相加**，如需合计会先查询并标注汇率

### 地点优先发现

当用户说"我想去福冈/大阪/东京/札幌"时，技能会自动进入地点优先模式：

- 将该城市映射到主要演出场馆（Zepp、Arena、Hall 等）
- 搜索官方渠道中该城市/地区的演出信息
- 按日期匹配度、乐队热度、票务可获性、场馆便利性综合排序推荐
- 若该城市无演出，主动推荐附近城市或替代日期
- 覆盖东京、大阪、名古屋、福冈、札幌、广岛、仙台等主要城市

### 灵活输出格式

| 格式 | 使用场景 | 扩展名 |
|------|----------|--------|
| Markdown | 默认，适合文本密集型报告 | `.md` |
| HTML | 含大量图片（特典图等）或需要丰富样式 | `.html` |
| PDF / Word | 用户明确要求时调用对应技能 | 按需 |

文件命名采用动态规则：`[乐队或活动名]_[日期或时间段]_[报告类型].[扩展名]`

## 文件结构

```
bangdream-live-planner/
├── SKILL.md      # 核心技能定义：工作流、规则、输出规范
├── reference.md  # 参考数据：搜索策略、乐队信息、输出模板
└── examples.md   # 示例输出：展示最终报告格式
```

## 使用方式

将本技能安装到 QoderWork 后，当提到以下关键词时会自动触发：

- BanG Dream、Bushiroad、Live 演出
- 演出日程、巡演安排、粉丝见面会
- 演唱会旅行规划、机票预订
- 特典（bonus goods）查询

也可以主动询问，例如：

- "端午节有三天假期，帮我看看 BanG Dream 的 Live"
- "请一天假去看 Roselia 的 10th live，帮我安排"
- "告诉我 MyGO!!!!! 最新一场演出的歌单和特典信息"
- "我想去日本看 Ave Mujica 的 FINAL，帮我规划行程和机票"
- "国庆想去大阪，那几天有 BanG Dream 的 Live 吗？"
- "打算春节去福冈，帮我看看附近有什么演出"
- "下个月想去札幌，有什么 Live 可以顺便看吗？"

## 已知乐队

BanG Dream 企划涵盖以下主要乐队：

- **Poppin'Party** — 初代乐队
- **Roselia** — 哥特摇滚风格
- **RAISE A SUILEN** — 电子摇滚 / DJ 风格
- **Morfonica** — 小提琴为核心
- **MyGO!!!!!** — 情感摇滚 / 乐队关系复杂
- **Ave Mujica** — 面具乐队 / 暗黑奇幻风格
- **夢限大MewType（夢限大みゅーたいぷ）** — 虚拟与现实结合

相关企划：
- **D4DJ** — Bushiroad 旗下 DJ 企划
- **Girls Band Cry** — 东映 3DCG 摇滚动画（风格相近，非同一企划）
